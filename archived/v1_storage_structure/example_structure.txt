## Problems with V1:
1. Type-first organization (audio/, transcripts/) splits related files
2. Inconsistent naming: some use video_title, others use video_id
3. No channel-level metadata
4. No processing completion markers
5. Difficult to see all files for one video at a glance

## V2 Improvements:
- Channel → Video hierarchy (everything for one video in one place)
- Consistent human-readable filenames using video_title
- Channel and video index files
- Processing completion markers
- Better metadata organization
